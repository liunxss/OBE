# 课程目标达成度分析系统

基于 Streamlit + Plotly 的交互式 Web 应用，
配套论文《面向工程教育认证的课程目标达成度矩阵计算模型设计与实践》。

## 快速启动

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动应用（浏览器会自动打开）
streamlit run app.py
```

默认访问地址：http://localhost:8501

## 文件说明

| 文件 | 说明 |
|------|------|
| `app.py` | Streamlit 主应用（含计算逻辑 + 可视化） |
| `course_attainment.py` | 独立计算脚本（批处理/命令行使用） |
| `requirements.txt` | Python 依赖列表 |

## 输入格式

### 得分表（Score Sheet）
| 序号 | 学号 | 姓名 | 题1 | 题2 | … |
|------|------|------|-----|-----|---|

### 分值分配表（Score Distribution Matrix）
| 题号 | 满分 | 子目标1 | 子目标2 | … |
|------|------|---------|---------|---|
> 校验：满分 = 各子目标分值之和

### 目标权重表（Objective Weight Table）
| 子目标 | 权重 |
|--------|------|
> 校验：权重之和 = 1.0

## 功能说明

- **多评估项**：支持任意数量的评估项（考试、作业、实验等），动态添加删除
- **动态权重**：加载目标权重表后，可通过滑块实时调整各子目标权重，结果即时更新
- **动态目标数**：子目标数量完全由分值分配表决定，不写死
- **阈值可调**：达成度合格阈值默认0.6，可滑动调整
- **交互图表**：全部使用 Plotly，悬停显示数值，支持缩放
- **一键导出**：结果 Excel + 全部图表 ZIP

## 局域网部署

让同院系教师通过浏览器访问：

```bash
streamlit run app.py --server.address 0.0.0.0 --server.port 8501
```

其他人访问 `http://你的IP:8501` 即可，无需安装任何软件。
